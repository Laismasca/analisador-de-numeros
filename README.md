# 📊 Analisador de Números

Projeto prático desenvolvido durante o curso de **JavaScript** do canal *Curso em Vídeo*, ministrado pelo professor Gustavo Guanabara.

## 🚀 Funcionalidades
O projeto permite ao usuário:
- Adicionar números entre 1 e 100 a uma lista.
- Impedir a adição de valores inválidos ou repetidos (se você implementou essa lógica).
- Exibir estatísticas ao clicar em "Finalizar":
  - Total de números cadastrados.
  - O maior valor informado.
  - O menor valor informado.
  - A soma de todos os valores.
  - A média dos valores.

## 🛠️ Tecnologias Utilizadas
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
